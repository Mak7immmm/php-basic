
Задание 1
Создайте переменную и присвойте ей любое целое число от 1 до 9.
Создайте вторую переменную и присвойте ей результат умножения числа 10 на случайное число от 1 до 3.
Подсказка: для получения такого случайного числа воспользуйтесь встроенной функцией в PHP — rand(1,3).
Выведите с помощью функции var_dump() обе переменные.
Напишите условную конструкцию if-else с тремя вариантами (включая вариант по умолчанию — else), которая будет по очереди проверять результат умножения первой переменной и второй по следующим критериям:
< 100,
< 200,
всё остальное.
В каждом случае выведите на экран с помощью конструкции echo строку «меньше 100», или «меньше 200», или третий любой вариант для конструкции else.
Подсказка: для вывода мы используем конструкцию «echo “”;», где в двойных кавычках пишем текст. 


Задание 2
Скопируйте код первого выполненного задания.
Замените присвоение конкретного числа первой переменной на случайное число из диапазона от 1 до 9 с помощью функции rand().
Подсказка: используйте синтаксис вызова этой функции, аналогичный заданию 1.
Создайте третью переменную перед конструкцией if-else и присвойте ей результат перемножения первой и второй переменных.
Замените конструкцию if-else на конструкцию выбора switch-case с выражением, равным логическому значению true.
В качестве проверочных значений (case) используйте диапазоны, в которые может попадать третья переменная:
>= 0, < 100;
>= 100, < 200;
>= 200, < 300
Подсказка 1: в качестве проверочных значений нужно использовать не конкретное число или переменную, а результат сравнения переменной с числами диапазона из этого задания.
Подсказка 2: чтобы объединить два условия (получить диапазон), используйте оператор «Логическое И».
Для каждого случая case выведите на экран с помощью конструкции echo сообщение 1, 2 или 3 соответственно.


Задание 3
Скопируйте код из второго выполненного задания.
После строки, где вы создаёте третью переменную, добавьте ещё одну строку, в которой к этой третьей переменной прибавьте случайное число от 0 до 100.
Условие: для операций суммы используйте запись инкремента, но не на единицу, а на абстрактное число
В конструкцию выбора допишите случай (case), который будет выполняться по умолчанию, если проверка не совпала ни с одним из трёх диапазонов, и выведите на экран в этом случае какой-то отличающийся текст (любой осмысленный).


Задание 4
Создайте переменную и присвойте ей случайное значение — либо 0, либо 1 — по аналогии с предыдущими заданиями, через функцию rand(). Выведите это значение на экран.
Создайте вторую переменную и присвойте ей значение — либо null если значение первой переменной равно 0, либо случайное число от 1 до 3 в противном случае. Выведите это значение на экран.
Условие: присваивать значение второй переменной нужно через тернарный оператор.
Создайте конструкцию выбора, где:
в виде выражения используется вторая переменная;
в виде значений (case) используются следующие:
null,
1,
case по умолчанию.
В каждом конкретном случае (case) выведите на экран какой-то осмысленный текст.
После конструкции выбора выведите на экран с помощью функции var_dump() значение логического типа, которое будет показывать, установлено ли значение для второй переменной или нет (то есть там пустое значение null).
Подсказка: для проверки установленности значения используйте функцию isset().
В конце создайте переменную, которой присвойте значение второй переменной, только если она НЕ null. В противном случае присвойте случайное значение от 20 до 30. И выведите значение этой переменной на экран.
Условие: нужно воспользоваться null-коалесцентным оператором.


Рекомендации по выполнению
Каждое задание делайте в отдельном файле: task_1.php, task_2.php и так далее.
Изучите возможности функции rand() в PHP.
Изучите конструкцию вывода данных на экран.
Изучите условные конструкции (+ тернарный оператор) и конструкции выбора:
мануал по elseif/else if;
мануал по switch;
сокращённая запись условия if else (тернарный оператор).
Вспомните операции инкремента/декремента.
Изучите материал по null-коалесцентному оператору.
