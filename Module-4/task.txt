Цели практической работы
Применить наиболее соответствующие заданию встроенные функции работы со строками в PHP и использовать все их особенности.
Применить функции работы с массивами, встроенные в PHP, наиболее лаконично решающие поставленные задачи.


Что нужно сделать
Задание 1
Дана строка: «Лето пришло».
Присвойте указанную фразу в виде значения переменной.
Найдите в этой фразе пробел (с помощью функции поиска подстроки в строке).
Присвойте новой переменной часть строки до пробела.
Выведите полученную переменную на экран.
Присвойте ещё одной новой переменной часть строки после пробела (с помощью функции разбиения строк на части и созданной выше переменной).
Выведите на экран последнюю переменную.
Задание 2
Дана строка с повторяющимися словами: «Я люблю море. Я лечу на море. Я умею плавать в море. Какое чистое море! Хочу на море. Завтра поедем на море».
Присвойте указанную фразу в виде значения переменной.
Найдите в ней повторяющееся слово и присвойте его в виде строки в другую переменную.
Замените повторяющиеся слова в указанной фразе на те же самые (с помощью функции для замены подстрок в строках), перевёрнутые в обратном порядке с помощью функции strrev. Подсказка: для переворачивания строки можно использовать встроенную функцию strrev. Условие: в функции замены подстрок нужно использовать созданные переменные.
Задание 3
Создайте переменную с ассоциативным массивом из трёх элементов, где ключами будут латинские буквы a, b и c, а значениями — любые строки на ваше усмотрение.
Присвойте новой переменной значение в виде простого массива, являющегося набором ключей массива из первой переменной. Подсказка: используйте одну из функций разбиения массива на ключи/значения.
Создайте третью переменную, значением которой должна быть строка, состоящая из значений второй (простого массива), разделённых пробелами. Подсказка: используйте функцию преобразования массива в строку, изученную в этом модуле.
Выведите на экран значение третьей переменной с помощью конструкции echo.
Задание 4
Создайте переменную, в которую положите ассоциативный массив из пяти элементов. Причём ключи должны быть уникальными, а как минимум два значения должны повторяться и идти НЕ подряд.
Посчитайте количество элементов в массиве с помощью функции определения размера массива и присвойте результат работы этой функции в новую переменную. Выведите её на экран.
Переверните массив с помощью функции замены ключей на значения, а значений на ключи и положите результат в ещё одну переменную.
Снова определите размер полученного массива и положите это значение в очередную переменную. Также выведите её на экран.
Выведите на экран результат сравнения переменной с размером первого массива и переменной с размером второго массива. Подсказка: для сравнения используйте оператор «не равно».
Задание 5
Скопируйте код предыдущего задания в новый файл с этим заданием.
Возьмите значения массива из пункта 3 предыдущего задания с помощью одной из функций разбиения массива на ключи/значения и положите результат в новую переменную.
Добавьте к полученному массиву дополнительный простой массив из одного (!) элемента, которого нет среди уже существующих элементов, а результат сохраните в новую переменную. Подсказка: используйте функцию объединения массивов.
Объедините полученный простой массив на предыдущем шаге со значениями массива, созданного в пункте 1 предыдущего задания. Подсказка: используйте функцию взятия значений массива.
Полученный объединённый массив присвойте переменной и выведите на экран.


Рекомендации по выполнению
Каждое задание делайте в отдельном файле: task_1.php, task_2.php и так далее.
Изучите все функции работы со строками и массивами, описанные в этом модуле.
