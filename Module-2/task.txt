Задание 1
Создайте две переменные с типами int и float и присвойте им значения, соответствующие этим типам.
Создайте третью переменную типа int и присвойте ей произвольное значение.
Выведите результат работы функции var_dump() от сравнения суммы первых двух переменных и третьей через оператор «меньше или равно».


Задание 2
Создайте две числовые переменные и задайте им значения 76 и 8 соответственно.
Выведите на экран результат арифметической операции «остаток от деления» между первой и второй переменной.


Задание 3
Создайте две числовых переменных, присвойте им значения 2 и 3 соответственно.
Сравните эти переменные, предварительно используя нужную форму инкремента для первой переменной, через оператор «больше или равно» так, чтобы результатом функции var_dump() было значение true.


Задание 4
Создайте две числовые переменные, присвойте им значения, равные 10.
Сравните эти переменные, предварительно используя нужную форму декремента для второй переменной, через оператор «не равно» так, чтобы результатом функции var_dump() было значение false.


Задание 5
Создайте переменную $a.
Присвойте переменной $a такое значение, чтобы конструкция
var_dump(isset($a) != true);
выдавала бы в качестве результата false.
Присвойте переменной $a другое значение, чтобы конструкция
var_dump(is_null($a) != false)
выдавала бы в качестве результата true.
